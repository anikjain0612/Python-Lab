Introduction to Python programming and installation of Jupyter Notebook using Anaconda

---

## Aim
To study the basics of the Python programming language, understand the Python execution environment, install Python and Jupyter Notebook using the Anaconda distribution, and explore various Python IDEs including Google Colab.

---

## Objectives
- To understand Python features and applications  
- To install Python and Jupyter Notebook using Anaconda  
- To explore Python IDEs and execution environments  
- To execute basic Python programs using Jupyter Notebook and Google Colab  

---

## Theory

### 1. Introduction to Python
Python is a high-level, interpreted, and object-oriented programming language. It is widely used for data analysis, machine learning, automation, and scientific computing due to its simplicity and readability.

---

### 2. What is Anaconda?
Anaconda is a free and open-source distribution of Python and R, mainly used for data science, machine learning, and scientific computing.

#### Advantages of Anaconda
- Comes with Python, Jupyter Notebook, Spyder, and essential libraries pre-installed  
- Provides easy package and environment management  
- Avoids dependency and compatibility issues  
- Suitable for beginners as well as professionals  

---

### 3. Installation of Anaconda and Jupyter Notebook

#### Step 1: Download Anaconda
1. Visit the official Anaconda website: https://www.anaconda.com  
2. Click on Download and select Anaconda Individual Edition  
3. Choose the Python 3.x version suitable for your operating system  

After installation, Jupyter Notebook can be launched directly from Anaconda Navigator.

---

## Python IDEs

| IDE | Description |
|----|------------|
| IDLE | Basic Python IDE |
| Spyder | Scientific Python IDE (included with Anaconda) |
| PyCharm | Professional Python IDE |
| VS Code | Lightweight and extensible code editor |
| Jupyter Notebook | Interactive environment for execution and analysis |

---

## Google Colab
Google Colab is a cloud-based Python environment provided by Google that allows users to write and execute Python code in the browser.

### Advantages
- No local installation required  
- Free access to GPU and TPU  
- Easy collaboration and sharing  
- Runs entirely in a web browser  

### Steps to Use Google Colab
1. Visit https://colab.research.google.com  
2. Sign in using a Google account  
3. Create a new Python notebook  

---

## Conclusion
Python provides a flexible and powerful programming environment. Tools such as Jupyter Notebook and Google Colab simplify the learning process and support interactive execution and visualization, making Python suitable for beginners and advanced users alike.
